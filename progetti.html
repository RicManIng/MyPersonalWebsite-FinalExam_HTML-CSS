Controllore Banco Test

Zapi - 2022/attuale

Skills :

  • C embedded
  • microcontrollori Microchip
  • MPlabXIde
  • FreeRTOS

Descrizione progetto :
Sviluppo di un software di controllo per l'esecuzione di test di durata sugli impianti cliente. Il controllore deve comandare l'impianto utilizzando il protocollo CanOpen proprietario ed inoltre deve verificare lo stato di sicurezza del test, intervenendo nel caso in cui si verifichino delle anomalie. Banco test hardware in the loop

Zapi - 2023/attuale

Skills :

  • Python
  • Harware in the loop
  • Linux embedded
  • Raspberry Pi

Descrizione progetto :
Sviluppo di un banco Hardware-in-the-loop per i test su software cliente.
Il banco è stato sviluppato utilizzando dispositivi interni controllati da una logica scritta in Python ed eseguita su un Raspberry Pi in ambiente Linux.
L'obiettivo era quello di garantire una risoluzione sulle variabili di controllo più elevata rispetto alla tolleranza accettata dalle specifiche della norma in modo tale da poter verificare le funzioni di sicurezza interne all'impianto.

Pick and Place con sistema di visione artificiale

K-loops - 2022/2023

Skills :

  • Python
  • Visione Artificiale (Open3D - OpenCV)
  • Manipolazione di pointcloud
  • Depth Camera (IntelRealSense)
  • robot KUKA
  • KUKA KRL
  • Visual Studio Code

Descrizione progetto :
Sviluppo di una libreria in Python che sfrutta le potenzialità di una depth camera come la IntelRealSense D435 per individuare oggetti nello spazio ed ottenerne coordinate cartesiane e orientamento.
Successivamente è stato richiesto di sviluppare un protocollo di avvicinamento per migliorare la stima iniziale della distanza dell'oggetto.
Sfruttando questa libreria è stato infine possibile testarla con una semplice operazione di pick and place utilizzando un robot industriale a 6 assi.
Si è quindi calcolata la trasformata tra il sistema di riferimento oggetto e il sistema di riferimento robot così da ottenere le rotazioni e le traslazioni necessarie ad afferrare l'oggetto.

Linea di produzione automatica

Esame università - 2022

Skills :

  • RobotStudio
  • ABB RAPID
  • Solidworks
  • LeanCost

Descrizione progetto :
Costruzione simulata di un'intera linea di produzione e verniciatura per quadri di arredo in metallo. Il progetto riguarda la produzione su commissione di design personalizzati di ritagli su metallo eseguiti da robot industriali a 6 assi e successiva verniciatura.
Si sono prodotti i disegni al CAD con Solidworks da introdurre nel workspace e successivamente si sono stimati i tempi e i costi di produzione sia del prodotto che delle apparecchiature necessarie alla produzione.
Le tempistiche sono state riportate all'interno di Robotstudio simulando un'intero ciclo di processo produttivo e scrivendo il software per i Robot ABB utilizzando il linguaggio proprietario RAPID.
